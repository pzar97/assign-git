What is a gitignore file and what should it include for a Java project?
- It is a plain text file that contains a pattern for files or directories to ignore.
- The files gitignore file should include for a Java project are-
# Compiled class file
*.class

# Log file
*.log

# BlueJ files
*.ctxt

# Mobile Tools for Java (J2ME)
.mtj.tmp/

# Package Files #
*.jar
*.war
*.nar
*.ear
*.zip
*.tar.gz
*.rar

# virtual machine crash logs, see http://www.java.com/en/download/help/error_hotspot.xml
hs_err_pid*

What does fast-forward in Git mean?
- Fast-forward in Git is used to merge the current branch to target branch by integrating the histories of both the branches.

Why should you have different branches?
- There should be different branches while working in Git because it isolates the team members work from other members and lets them maintain an independent line.

In a project how many remote Git repositories do you usually have?
- There can be as many remote Git repositories the team wants in a project. But there can be only one remote repository that has the name "origin".
